# Week 1 AI Assignment: Live API Data Processor

## Project Description
This repository contains a Python-based automation tool developed for the Week 1 AI Assignment. The application interfaces with a free, public API to retrieve live data payloads, safely parse the JSON structural items, clear empty nodes, and print a human-readable, well-formatted terminal layout feed.
